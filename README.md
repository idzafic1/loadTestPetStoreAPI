# Vizualizacija Load Testiranja - Petstore API

Ovaj repozitorij je namijenjen isključivo za **grafički i vizuelni prikaz rezultata load testiranja** izvršenog na javnom Petstore API-ju.

## Svrha
Glavni cilj projekta je pružiti jasan uvid u performanse sistema kroz vizualizacije dobivene korištenjem alata **k6**.

Repozitorij obuhvata:
*   **Grafičke prikaze**: Detaljni grafikoni koji prikazuju ključne metrike poput vremena odziva (latency), propusnosti (throughput) i stope grešaka.
*   **HTML Izvještaje**: Generisani izvještaji koji omogućavaju jednostavnu analizu rezultata testiranja bez potrebe za ponovnim pokretanjem testova.
*   **Analizu performansi**: Vizuelni pregled ponašanja API-ja pod različitim nivoima opterećenja (Smoke, Stress, Spike i Load testovi).

## Alat
Sva testiranja su provedena korištenjem **k6**, modernog alata za testiranje performansi otvorenog koda.
